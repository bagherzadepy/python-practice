## راه اندازی و اجرا

1. کلون کردن مخزن:
```bash
git clone https://github.com/bagherzadepy/python-practice.git
```
2. ورود به پوشه پروژه
```bash
cd python-practice
```
3. اجرای فایل:
```bash
python day1.py
```
