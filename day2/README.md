# Python Practice - Day 2

1. **مدیریت نمرات** (`grades.py`)
    - محاسبه بالاترین نمره، پایین ترین نمره، معدل

    - مدیریت سبد خرید با دیکشنری

3. **جدول ضرب** (`multiplication.py`)
    - چاپ جدول ضرب 1 تا 10
